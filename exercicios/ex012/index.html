Inserindo videos hospedados localmente

Formatos de videos

Os mais utilizados e mais suportados nos navegadores são:

  • MP4/M4V
  • OGV
  • WEBM

Inserir os videos

Para inserir os videos, localmente, não é muito diferente dos audios.
Utiliza-se a tag <VIDEO> junto de alguns parametos: POSTER; CONTROLS.

Segue exemplo: 

    <video width="700" poster="imagens/lcx.jpg" controls >
    <source src="midia/Meuvideo.webm" type="video/webm">
    <source src="midia/meuvideo.mp4" type="video/mp4">
    <source src="midia/meuvideo.ogv" type="video/ogg">
    <p>Seu navegador não tem compatibilidade com reprodução de videos.</p>
    </video>
        


    Onde:
    VIDEO tag aberta para condicionar que será selecionado um arquivo de video.
    POSTER Indica uma thumbnail(CAPA) para o video.
    CONTROLS indica os controles do videos, como: PLAY; VOLUME; VELOCIDADE DE REPRODUÇÃO;
