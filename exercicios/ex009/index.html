Trabalhando com listas

Listas Ordenadas

A ordem faz diferença

Utiliza-se a tag <OL>

Pode-se utilizar um paramentro TYPE, que muda o tipo da lista, como: Número; letra;
Exemplo: 1. A. a. I. i.

Pode também escolher por qual número/letra começar, pelo parametro START

Exemplo de uso dos parametros: 

    <OL type="I" start="3">
        <LI> III. ir para academia
    </ol>

A tag <LI> não precisa ser fechada, porém o VisualCode fecha ela pelo atalho

Segue exemplo de lista:

  1. ir para academia
  2. estudar
  3. ir para o estagio
  4. estudar quando voltar
  5. Procurar casas a venda
  6. Estudar sobre o negocio de escolas infantis

Listas não ordenadas

A ordem não faz diferença

Utiliza-se a tag <UL>

Pode-se utilizar um paramentro TYPE, que muda o tipo de "ponto"
Tem 3 tipos de TYPE no UL, estes são: SQUARE; DISC; CIRCLE

Exemplo de uso do parametro: 

    <OL type="circle">
        <LI> • Pão
    </ol>

Segue exemplo de lista:

  • Pão
  • alface
  • tomate
  • ovo
  • cebola

Juntando listas versão 1.0

Dá pra juntar as listas, literalmente fazer uma dentro da outra.

Segue exemplo:

Desenhos favoritos

  • Antigos
    1. Scooby-Doo
    2. Simpsons
    3. SouthPark
    4. Jhonny Bravo
  • Atuais
    1. Apenas um show
    2. hora de aventura
    3. sei lá, ficou uma merda os desenhos

Lista definições

A lista de definição é como um dicionário, tem o termo e a descrição. Utiliza-se a tag <DL> para criar a lista. <DT> para criar o termo, e <DD> para criar a descrição.

Exemplo: 

    <DL>
        <DT>HTML</DT>
        <DD>Linguagem de marcação para  criação do conteúdo de um site.</DD>
    </DL>

Segue lista exemplo:

HTML
Linguagem de marcação para criação do conteúdo de um site.
CSS
Linguagem de marcação para criação do desing de um site.
JavaScript
Linguagem de programação para criação de interatividade de um site.
