Usando links externos

Para usar um link em um techo é simples, basta usar a tag <A>
Porém, se link for externo ao seu dominio é recomendavél utilizar os paramentos Target="_blank" e rel="external"

Onde:
target="_blank" indica o link a executar em outra "aba"
rel="external" dá significado de link externo ao do propio site

Exemplo:

Link externo:
Você pode acessar o meu perfil no instagram

Link no proprio site:
Você pode acessar meu canal no youtube

Usando link Internos

Link interno vai fazer a relação com outra pagina do proprio servidor

Exemplo:
Essa aqui é a minha primeira pagina, se você quiser acesar a minha segunda pagina clique aqui!

Paginas em outras pastas

Existe a opção de acessar outra pagina em outra pasta no proprio servidor

Exemplo:
Veja também minha terceira pagina, que fala sobre drogas

Links para dowloads

Dependendo do navegador e do arquivo, ao invés de ele baixar imediatamente o arquivo, ele apenas abre, tipo arquivos PDF

Não tem muita diferença para adicionar um link para dowload, o que vai diferenciar de um link normal vai ser os parametros colocados: DOWLOAD e TYPE.

Onde:
DOWLOAD="certificadoMysql.pdf" indica que o link seguido é para dowload
TYPE="application/pdf" indica o tipo de arquivo que será gerado para dowload

Exemplo:

  • Baixar livro em PDF
  • Livro compactado em ZIP
