Imagens dinamicas

O objetivo destas imagens, é que o proprio site se arrume para tela em que esteja sendo aberto

Para isso, é utilizada a tag <PICTURE>. Dentro desta tag é utilizada a tag de por imagens(<IMG>)
Dentro da tag <PICTURE> adicione a tag <SOURCE:MEDIA:TYPE>, acima da imagem principal, que geralmente é maior formatação. Segue exeplo:

<source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">

Onde:
MEDIA indica que se o navegador tiver menos pixels, ele deve exibir a imagem do parameto a seguir.
SRCSET Indica qual imagem ele tem que procurar caso o navegador tiver menos pixels que o declarado acima.
type Indica o tipo de imagem(PNG; JPEG).

Reproduzindo audios

Testando reproduzir audios em HTML5

Para adiconar um audio no site, basta utilizar a tag <AUDIO>, dentro dela faça a busca do documento. Segue exemplo:

<audio src="midia/Lucrixx.mp3" controls autoplay> </audio>

Onde:
SRC Faz a busca de onde o audio está, recomendavél está em uma pasta diferente do index.
CONTROLS Faz aparecer os controles do audio, como PLAY; VOLUME; VELOCIDADE DE REPRODUÇÂO;
AUTOPLAY Faz o audio iniciar junto com o site, em alguns navegadores esse parameto não funciona.

Muitos navegadores não tem compatibilidade com todos os formatos
Para isso é utilizada outra tag dentro de <AUDIO>. Segue exemplo:

<audio controls>
    <source src="midia/Lucrixx.mp3" type="audio/mpeg">
    <source src="midia/Lucrixx.mp3" type="audio/ogg">
    <source src="midia/Lucrixx.mp3" type="audio/wav">
    <p>Infelizmente seu navegador não consegue reproduzir audio. <a href="midia/Lucrixx.mp3"> Clique aqui para baixar o arquivo MP3 </a></p>
</audio>


    Onde: 

    SRC Faz a busca de onde o audio está. 

    TYPE Diz qual o tipo do documento



A diferença é que caso o navegador não abra em mp3, ele tenta a outra opção. No caso somente 3 opções. 

Se nenhuma das opções abrir ele deve mostrar o texto e o link para dowload do arquivo.

Existem os parametos:
PRELOAD: AUTO; METADATA; NONE;(AUTO: Só vai considerar que o site carregou completamente quando o audio carregar completamente; METADATA: Carrega só o nome do arquivo, data, duração; NONE: Não carrega nada)
AUTOPLAY: Reproduz o audio junto com a inicialização do site
LOOP: o nome diz
