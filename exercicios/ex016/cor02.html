Como criar um degradê em CSS

Dentro do <HEAD> é criada a TAG <STYLE>.

Dentro da tag <STYLE> coloca-se o body e adiciona o parametro"background-image: linear-gradient (to left/right/top/bottom white, black)"

Onde:
background-image é a propriedade
linear-gradient é função interna
"To" left/right/top/bottom é o lado por onde começa e termina as cores escolhidas.
OBS: é possivél trocar o angulo do gradiente. Utilizando "45deg"(45 seria o numero de graus. 'DEG' é abreviação para degraus) ao invés de "to left"

Exemplo:
body{ background-image: linear-gradient(to left, rgb(236, 209, 53), rgb(207, 43, 212)); }
