Trabalhando com fontes

Aplicando as fontes

Para utilizar as fontes é facil, coloca-se dentro do arquivo CSS na sintaxe BODY a propriedade font-family. Exemplo no codigo:

    body{
        font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    }

O problema maior vem que em alguns navegadores, ou displays como televisões, não possuem todas as fontes intaladas. Para resolver isso é utilizada uma combinação de fontes, Chamada "Safe Fonts"

Essa combinação faz com que, caso o navegador não encontre a primeira fonte, ele selecione a segunda da fila, assim sucessivamente.

Tamanho das fontes

Existem dois tipo de medidas de fontes:

  1. Medidas Absolutas
  2. Medidas Relativas

1. Medidas Absolutas

  • cm (Centimetros)
  • mm (milimetros)
  • in (polegadas)
  • px (pixels)
  • pt (ponto)
  • pc (paica)

Medidas Relativas

  • em (Tamanho da letra M, maiuscula, da fonte atual)
  • ex (altura X)
  • rem (root do body)
  • vw (largura da viewport)
  • vh (altura da viewport)
  • porcentagem

A W3 ORG, recomenda utilizar somente PIXEL e EM como tamanho das fontes em sites.

Exemplo no codigo:

    body{
        font-size: 1em;
    }

Peso da fonte

Para começar, o peso da fonte tem que ser testado de acordo com a fonte selecionada. Pois nem todas as fontes tem certas variações

Para utilizar usa-se a propriedade "font-weight"

Existem dois tipos de pesos:

  1. Leterários
  2. Numéricos

1. Literários

  • Lighter
  • normal
  • bold
  • bolder

2. Numéricos

Começa em 100 e termina em 900. A aplicaçõa é de 100 em 100.
Exemplo:
font-weight: 100;
font-weight: 300;
font-weight: 800;

Exemplo no codigo:

    body{
        font-weight: normal;
    }

Uso do estilo da fonte

O font-style é basicamente para indicar se a fonte a ser usada será "Normal" ou "Itálico". Porém nem todas as fontes tem itálico

Exemplo no código:

    body{
        font-style: italic; 
    }

Sublinhado

Para o uso do sublinhado em CSS, utiliza-se a propriedade "Text-Decoration". Existem dois valores mais utilizados, o underline e line-through.

Exemplo no código:

    body{
        text-decoration: underline; 
    }


Propriedades ShortHand

Propriedades abreviadas (tradução literal: Propriedades abreviadas) são propriedades de CSS que permitem que você defina os valores de outras propriedades de CSS simultaneamente. Usando propriedade taquigráfica, um desenvolvedor web pode escrever folhas de estilo mais concisas e legíveis, economizando tempo e energia.

Exemplo:
Uma formatação normal de código CSS seria assim: 

    body{
        font-family: Sans-serif;
        font-weight: normal;
        font-size: 3em;
        font-style: italic; 
    }

Agora feito como um Shorthand Font, ficaria assim: 

    body{
        font: italic normal 3em Sans-serif
    }

O ShortHand faz uma junção de todas as propriedades em uma só. Porém existe um padrão a ser seguido: Style; Weight; Size; Family.
