Seletores personalizados

Utilizando IDs

As ids são uma forma de identificar um elemento, e devem ser ÚNICAS para cada elemento. É como se fossem impressões digitais de nossos dedos ou RGs. Através delas, pode-se atribuir formatação a um elemento em especial.

No código fonte, escolha o elemento que você quer aplicar a personalização. E na CSS aplique o desing.

Exemplo no código:

        No HTML: 

        <H1 ID="primeiro">Exemplo</H1>
            
        Na CSS: 

        H1#primeiro{
            Color: Red;
        }

*Obs: Lembrando que as IDs são identificadas por #

Utilizando Class

As classes são uma forma de identificar um grupo de elementos. Através delas, pode-se atribuir formatação a VÁRIOS elementos de uma vez.

Diferente das IDs, as classes podem ser utilizadas mais de uma vez, e por varios seletors.

Exemplo no codigo:

        No HTML: 

        <H1 class="basico">CSS Básico</H1>
            
        Na CSS: 

        h2.basico{
            color: red;
        }

*Obs: Lembrando que as Class são identificadas por .

Pseudo Classes

Uma pseudo-classe CSS seleciona um elemento HTML de acordo com o estado deste elemento. Este estado pode estar relacionado com algum evento que ocorra no elemento (como um input do usuário ou uma interação com o mouse), ou então com a relação de uma tag com suas tags filhas.

As pseudo-classes iniciam com símbolo : (dois pontos) e geralmente são aplicadas a algum seletor CSS mais simples, como um seletor de classe ou um seletor de tag, permitindo formatações mais específicas de acordo com o estado do elemento.

Hover

A pseudo-classe "Hover" é a que mais pode ser utilizada para essa explicação.

O Hover permite que um elemento tenha seu estilo alterado através da ação do usuário, quando o próprio passa o cursor do mouse sobre o elemento.

Exemplo no código:

        div:hover{
            background-color: blue;
            color: white;
        }

Nesse exemplo do códgio acima, faz com que quando o mouse passar por cima de uma div, fique azul e as cores fontes brancas.

Podemos fazer com textos também(Passe o mouse aqui)

Para fazer esse exemplo do texto acima, utilizei:

        p#ex-div:hover{
            color: blue;
            text-decoration: underline;
        }

*Obs: Isso não transforma a div, ou textos em links.
